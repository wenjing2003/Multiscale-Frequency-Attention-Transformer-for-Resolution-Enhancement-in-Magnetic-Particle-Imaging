# Multiscale-Frequency-Attention-Transformer-for-Resolution-Enhancement-in-Magnetic-Particle-Imaging
We propose a Multiscale Frequency Attention Transformer (MFAT) model, which integrates a frequency-domain discriminative feedforward network (DFFN) with a multiscale attention block (MAB) to form an end-to-end framework for recovering high-resolution MPI images from low-resolution reconstructions.
